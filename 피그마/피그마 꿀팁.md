# 피그마 공부

## 피그마 필수 단축키

|                 |                    |
| --------------- | ------------------ |
| 기능 검색       | `Ctrl + /`         |
| 오브젝트 복제   | `Ctrl + O`         |
| 투명도 조절     | `숫자 0~9`         |
| 그룹            | `Ctrl + G`         |
| 그룹 해제       | `Ctrl + Shift + G` |
| 눈금자          | `Shift + R`        |
| 컴포넌트 만들기 | `Ctrl + Alt + K`   |
| 했던 작업 반복  | `Ctrl + D`         |
